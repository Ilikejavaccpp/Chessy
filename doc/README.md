# README.md

## What is Chessy?

Chessy is a free to play, no subscription, chess app built for learning, especially intermediates or people who want to level up their chess skills. It currently is a simple chess app. In the future, we **will** include the following:
* Chess engine (stockfish) + Game report/review (free!)
* Learn mode (where you can take back turns and play with your friend to learn together)
* Database for both openings, positions and matches
* Online and import (PGN) from (Chess.com)[https://www.chess.com] or Lichess

## Why was Chessy Made?

Chessy is made because let's be real... There is only **ONE** game report in (chess.com)[htttps://www.chess.com] and it also asks us for a diamond, silver, etc. subscription. Also... everything is worse on mobile. So, as a chess player myself, I decided to put an end to this and make this app.

## Installation

Installation is pretty easy. You have three options...
- Build via git and the native `build.sh`
- Get the executables from the bundles specifically for your os (AMD coming soon !!)
- Download the executable from (the official website)[https://yapp_colection/chessy.com]
